#### 项目的部署：
>1，git clone git@github.com:chenalian/Blog-SpringBoot-Vue.git

>2.打开springboot-后端中的application.yml配置好数据库和redis，并将数据库文件导入

>3.打开vue-前端，用命令行打开,并输入命令
> ```
> npm install   //下载依赖
> npm run dev  //启动项目
> ```
